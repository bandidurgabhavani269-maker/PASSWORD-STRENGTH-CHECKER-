# 🔐 Password Strength Checker

## 📌 Project Overview
The Password Strength Checker is a simple cybersecurity project that evaluates the strength of a password based on predefined security rules. It helps users create strong passwords and improve online account security.

---

## 🎯 Objective
- Evaluate password strength.
- Guide users to create secure passwords.
- Improve cybersecurity awareness.
- Reduce the risk of password-related attacks.

---

## 📖 Introduction
Passwords protect personal accounts and sensitive information from unauthorized access. Weak passwords are easy to guess or crack using hacking techniques. This project checks the strength of a password and classifies it as **Weak**, **Medium**, or **Strong** based on security requirements.

---

## 🔍 Password Strength Criteria
A strong password should include:
- Minimum 8–12 characters
- At least one uppercase letter (A–Z)
- At least one lowercase letter (a–z)
- At least one number (0–9)
- At least one special character (@, #, $, %, &, etc.)

### Example
- Weak: `12345`
- Strong: `A@b9#X2kL`

---

## ⚙️ Working
1. User enters a password.
2. The system checks:
   - Password length
   - Uppercase letters
   - Lowercase letters
   - Numbers
   - Special characters
3. The password is classified as:
   - Weak
   - Medium
   - Strong

---

## 💻 Sample Output

| Input | Strength |
|--------|----------|
| abc123 | Weak |
| Abc12345 | Medium |
| A@b9#X2k | Strong |
